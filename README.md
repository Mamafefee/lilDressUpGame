# lilDressUpGame!

A fun browser-based dress up game where you can mix and match outfits and accessories on a character! 👗

> 🎮 **[Play now →](https://mamafefee.github.io/lilDressUpGame/)**

![demo](https://github.com/user-attachments/assets/4b4edf10-c148-4ae6-bb86-d029c64a173b)

---

## ✨ Features

🎀 **6 categories** — each with 3 options to mix and match

🔊 Sound effects on every interaction

📦 Items loaded dynamically via JSON

---

## 🛠️ Built with

- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON + Fetch API

---

## 🚀 How to run locally

```bash
git clone https://github.com/mamafefee/lilDressUpGame.git
cd lilDressUpGame

# ⚠️ Open with Live Server (VS Code) — required for JSON fetch to work ⚠️

```
